# ServiceEngineeringDETI

__To run this project you need:__
* Python environment
* SQLite installed
* Flask framework, if you have pip installed you can do:
	* _$sudo pip install Flask_ 

__How to run:__
* First run the script to create the database 
	* _$sh createBD.sh_
* Run the python script
	* _$python run.py_
